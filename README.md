# 🧵 Sentiment Analysis of Customers of Textile Fashion Brands in Pakistan

This project analyzes customer sentiment toward leading Pakistani textile fashion brands using real social media comments. The focus is on uncovering key sentiment drivers, assessing the impact of sustainability campaigns, and generating actionable insights through data visualization and sentiment modeling.

## 📌 Objectives

- Collect authentic customer comments from platforms like Facebook, Instagram, and Twitter.
- Preprocess text data to clean and normalize input for accurate analysis.
- Perform sentiment classification (positive, negative, neutral) using NLP techniques.
- Visualize sentiment trends and brand perception over time.
- Evaluate the impact of marketing efforts on customer attitudes.
- 
## 🛠️ Tools and Libraries

- **Data Collection**: Selenium, BeautifulSoup, Tweepy
- **Preprocessing**: NLTK (stopwords, lemmatization), regex
- **Sentiment Analysis**: TextBlob, VADER (SentimentIntensityAnalyzer)
- **Visualization**: Matplotlib, Seaborn, Plotly

## 📊 Key Results

- **Sentiment Distribution**:  
  - Positive: 54%  
  - Negative: 28%  
  - Neutral: 18%

- **Top Positive Drivers**: “High quality”, “eco-friendly”, “fast delivery”  
- **Top Negative Drivers**: “Late delivery”, “poor customer service”, “overpriced”

- **Marketing Impact**:  
  - *Sapphire’s* sustainability campaign increased positive sentiment from 48% to 61%.  
  - *Gul Ahmed* saw a complaint spike post-Eid due to late deliveries.

## 🗓️ Temporal Trends

Sentiment improved significantly during well-communicated sustainability campaigns but dropped when customer service or delivery performance declined near key shopping seasons like Eid.

## ✅ Conclusion

This project demonstrates the power of sentiment analysis for brand monitoring. Insights generated here can help textile fashion brands improve customer experience, align marketing with customer expectations, and boost loyalty through data-driven decision-making.
